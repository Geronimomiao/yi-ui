# API

# u-back-top props

属性 说明 类型 默认值
height 页面滚动高度达到该值时才显示u-back-top组件
bottom 组件距离底部的距离 Number 30
right 组件距离右部的距离 Number 30
duration 滚动动画持续时间，单位 毫秒 Number 1000

# u-back-top events

事件名 说明 返回值
on-click 点击按钮时触发 无
